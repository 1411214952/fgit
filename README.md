# fgit
A tool to replace your github URL to fastgit URL

## How To Useï¼Ÿ
### Linux
1. Clone fgit
2. `ln -s` fgit to `/bin/`
3. Use commands `fgit https://github.com/xxx/xxx.git ` to  use the fastgit.org clone repository

### Windows
1. Clone fgit
2. add fgit/bin to you PATH
3. Use commands `fgit https://github.com/xxx/xxx.git ` to  use the fastgit.org clone repository
